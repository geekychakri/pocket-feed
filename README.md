## Run Locally

Clone the project

```bash
  git clone https://github.com/GeekyChakri/pocket-feed.git
```

Go to the project directory

```bash
  cd pocket-feed
```

Install dependencies

```bash
  npm install
```

Create an .env file in root and add your variables

```
  AUTH0_SECRET=
  AUTH0_BASE_URL=
  AUTH0_ISSUER_BASE_URL=
  AUTH0_CLIENT_ID=
  AUTH0_CLIENT_SECRET=
  MONGODB_URI=
  TWITTER_BEARER_TOKEN=
```

Start the app

```bash
  npm run dev
```
